## 10. CONTENT BRIEF NOTES FOR EACH PAGE TYPE

### Model Range Pages (`/vehicles/[slug]`)
- **Primary KW:** `[model name] range` — use in H1, title tag, first paragraph
- **Secondary KWs:** `[model name] real world range`, `[model name] winter range`, `[model name] charging time`
- **Title format:** `[Model Name] Range: Real-World Range Calculator & Data [Year] | [SiteName]`
- **Meta desc format:** `What's the real range of the [Model Name]? Our calculator adjusts for temperature, speed, and driving style. EPA says [X] miles — owners report [Y] miles on average.`
- **H1:** `[Model Name] Range Calculator`
- **Required sections:** Interactive mini-calculator, Range by condition table, Real-world range data, vs Top 3 competitors, Charging guide, FAQ (6-8 Qs)
- **Schema:** Vehicle + Product + FAQPage + BreadcrumbList

### Comparison Pages (`/compare/[slug]`)
- **Primary KW:** `[model a] vs [model b]`
- **Title format:** `[Model A] vs [Model B]: Range, Charging, Price Compared [Year] | [SiteName]`
- **Required sections:** Side-by-side spec table, Range comparison chart, Charging speed chart, 5-year TCO comparison, Verdict, FAQ
- **Schema:** ItemList + FAQPage + BreadcrumbList

### "Best EV For" Pages (`/best-ev-for/[usecase]`)
- **Primary KW:** `best ev for [use case]` + variant: `[longest range/best towing/etc] electric car`
- **Title format:** `Best EVs for [Use Case] in [Year]: Data-Driven Rankings | [SiteName]`
- **Required sections:** Ranked list with data cards, Comparison table of top picks, Calculator CTA, FAQ
- **Schema:** ItemList + FAQPage + BreadcrumbList

### Tool Pages
- **Primary KW:** `ev [tool name] calculator`
- **Required:** Interactive tool above fold, results section, "how it works" explanation, FAQ, related tools in sidebar
- **Schema:** WebApplication + HowTo + FAQPage
